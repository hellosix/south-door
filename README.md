# South Door
站点收藏平台，支持 HTTP 和 TCP 代理。
> 如果该工具对您有帮助，可以给我们个 star 支持一下，谢谢！ ^_^
使用过程中如果有问题或者建议可以给我们提 Issue。

## 功能
* 网站收藏
* 网站代理，例如：服务部署到线上，而本地无法访问，这个时候可以在跳板机使用 HTTP 或 TCP 代理来访问
