# south-door
> 南天门站点管理平台，支持 tcp 代理功能

## 一、特性

#### 1 支持网页收藏功能，自动生成网页缩略图
#### 2 支持 tcp 代理，假设服务部署到线上本地无法访问可以这个时候可以在跳板机使用 tcp 代理来访问

## 二、使用方式
> 南天门主要包含来两个模块到功能：网页收藏、tcp 代理
